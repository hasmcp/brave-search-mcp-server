# brave

## Brave Search API provider for MCP Server

Get a Brave Search API Key from https://brave.com/search/api/. Create variable called `API_SEARCH_BRAVE_COM_API_KEY` and assign the Braver Search API Key as secret.

**Base URL:** `https://api.search.brave.com/res/v1`

**OAuth2:** ❌ Not available

**Documentation:** [https://brave.com/search/api/](https://brave.com/search/api/)

## Tools (1)

- [webSearch](tools/web-search.md) — Performs a web search using Brave Search API.
