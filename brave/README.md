# brave

## Brave Search MCP Server

Search the Web for various types of data via the Brave Search Official API.

## Providers

### [brave](brave)

## Brave Search API provider for MCP Server

Get a Brave Search API Key from https://brave.com/search/api/. Create variable called `API_SEARCH_BRAVE_COM_API_KEY` and assign the Braver Search API Key as secret.

#### Tools (1)

- [webSearch](brave/tools/web-search.md) — Performs a web search using Brave Search API.
